# 一、项目执行摘要

本项目聚焦AI创意内容生产赛道，打造一款**基于Excalidraw轻量化无限画布、融合Blender几何节点程序化编辑逻辑、集成全品类AI生成能力**的一站式创意设计平台。平台打破传统设计工具功能单一、操作复杂、创作链路割裂的痛点，以无限自由画布为载体，将AI文生图、图生图、文生视频、图生视频功能与节点化程序化编辑深度融合，实现从创意构思、素材生成、逻辑编辑、效果调试到成品输出的全流程可视化创作。

产品核心差异化在于，摒弃传统AI生成工具“一键生成、无法精细化调控”的弊端，借鉴Blender几何节点的模块化、参数化、程序化编辑思维，让普通用户无需专业建模、剪辑、设计技能，即可通过节点拖拽、参数微调、图层联动，精准控制AI生成内容的细节、风格、动态逻辑；同时依托Excalidraw极简、轻量、无限延展的画布特性，支持海量创意元素自由排布、无限拓展、实时编辑，适配图文设计、短视频创作、视觉创意、动态海报、概念设计等多场景需求。

项目旨在打造一款**低门槛、高自由度、强可控、全链路**的AI创意生产工具，服务个人创作者、自媒体从业者、中小设计团队、电商运营、学生群体等用户，填补市场“轻量化无限画布\+节点化AI精细化生成”的产品空白。

# 二、项目背景与行业分析

## 2\.1 行业发展现状

当前AI生成式技术高速普及，文生图、文生视频工具已成为创意行业主流生产力工具，AI创意市场持续爆发式增长。但现有产品存在明显两极分化问题：一是轻量化AI工具（如各类一键生成图片、视频小程序）操作简单，但缺乏精细化调控能力，生成效果随机性强、无法自定义创作逻辑，成品质量参差不齐；二是专业设计工具（Blender、PR、PS等）功能强大、可控性高，但上手门槛极高、操作复杂，且未原生集成AI全品类生成能力，创作链路繁琐。

同时，Excalidraw类无限画布工具凭借轻量化、自由拓展、可视化排布的优势，已广泛应用于思维导图、流程设计、创意构思场景，但目前仅支持基础图形绘制，无AI生成与动态创作能力；Blender几何节点的程序化编辑模式，是行业公认的精细化内容调控最优方案，但仅适配专业3D创作，未适配2D图文、短视频轻量化创作场景，受众范围极窄。

## 2\.2 行业痛点与用户需求

**核心痛点**：

- AI生成工具：一键生成自由度低，无法拆解创作逻辑、精细化调整细节，难以满足个性化创作需求；

- 传统设计工具：操作门槛高、学习成本大，AI能力适配性差，创作流程繁琐低效；

- 常规无限画布工具：仅支持静态图形绘制，无AI创意生成、动态视频创作能力，功能单一；

- 创作链路割裂：素材生成、画面排版、效果调整、动态制作需要切换多款工具，耗时耗力。

**用户核心需求**：低门槛上手、无限自由创作空间、AI全品类内容生成、可精细化调控创作细节、一站式完成图文\+动态视频创作。

## 2\.3 项目优势与市场机会

本项目精准契合行业缺口，将三大主流技术优势融合创新：以Excalidraw无限画布解决创作空间限制问题，以Blender几何节点逻辑解决AI生成不可控问题，以全品类AI大模型能力解决创意素材生产问题，实现“轻量化操作、专业化调控、全场景适配”的产品特性，覆盖大众轻量化创作与半专业精细化创作市场，具备极强的市场差异化竞争力。

# 三、产品核心功能设计

产品整体架构分为三大核心模块：**Excalidraw轻量化无限画布模块、Blender式几何节点程序化编辑模块、全品类AI大模型生成模块**，三大模块深度联动，支撑全流程创意创作。

## 3\.1 基础载体：Excalidraw风格无限画布

复刻Excalidraw核心优势，打造轻量化、高自由度、无限延展的可视化创作画布，作为所有创作内容的承载载体，核心特性如下：

- **无限自由延展**：无画布边界限制，支持无限放大、缩小、平移，可自由排布图片、图形、文字、视频、节点组等所有创作元素，适配大型创意构图、多元素组合创作；

- **轻量化极简交互**：延续Excalidraw手绘风、极简操作逻辑，无复杂冗余功能，拖拽、缩放、排布、编组操作流畅，低配置设备也可稳定运行；

- **多层级元素管理**：支持元素分层、分组、锁定、隐藏，支持画布内容实时保存、云端同步、版本回溯，杜绝创作内容丢失；

- **自由排版组合**：支持手绘图形、自定义文字、导入素材、AI生成素材自由拼接组合，适配创意草图、海报设计、场景构图、分镜脚本等静态创作场景。

## 3\.2 核心调控：Blender几何节点式程序化编辑系统

借鉴Blender几何节点的模块化、参数化、程序化核心逻辑，打造适配2D图文、短视频创作的可视化节点编辑系统，解决传统AI生成“不可控、难微调”的核心痛点，是产品差异化核心壁垒：

- **模块化节点拖拽编辑**：将AI生成、画面调整、特效渲染、动态参数、视频剪辑等功能拆解为独立节点，用户可通过拖拽节点、连线组合，自定义完整创作流程，无需代码、无需专业技术；

- **参数化精细化调控**：每个节点支持自定义参数，可精准调整画面风格、色彩、光影、构图、清晰度，视频帧率、运动轨迹、转场效果、时长等细节，实现AI内容的精准化定制；

- **程序化联动逻辑**：支持节点组嵌套、参数联动、批量生效，修改单一核心参数即可同步调整全局画面/视频效果，支持重复复用创作逻辑，大幅提升创作效率；

- **实时预览调试**：节点编辑过程中，画布实时渲染预览效果，支持随时修改、即时更新，告别传统工具“生成\-修改\-重新生成”的低效循环。

## 3\.3 核心能力：全品类AI大模型生成功能

深度接入主流AI大模型，在无限画布与节点系统的支撑下，实现**文生图、图生图、文生视频、图生视频**全功能落地，所有生成内容均可通过节点精细化调控：

### 3\.3\.1 AI文生图

用户在画布输入自然语言Prompt，结合风格、构图、分辨率、光影等节点参数，一键生成定制化图片，支持插画、写实、二次元、国潮、科幻等全风格创作，可通过节点微调细节、优化构图、修正瑕疵。

### 3\.3\.2 AI图生图

支持上传自定义素材、选取画布已有画面，通过节点设置风格迁移、画质增强、细节重构、比例调整等参数，实现图片二次创作、风格改造、高清修复，保留原图核心结构的同时智能优化效果。

### 3\.3\.3 AI文生视频

依托文本描述生成动态短视频，支持通过节点自定义视频时长、帧率、运动节奏、镜头角度、动态特效、背景音乐适配等参数，生成符合需求的动态创意视频，适配短视频、动态海报、创意短片场景。

### 3\.3\.4 AI图生视频

以画布静态图片、手绘构图、设计稿为基础，通过节点设置动态运动逻辑、镜头推拉、元素动画、转场效果，将静态画面一键转化为动态视频，实现静态创意到动态成品的无缝转化。

## 3\.4 辅助配套功能

- 素材库：内置海量免费模板、背景、贴纸、动态素材，支持自定义素材导入；

- 作品导出：支持图片（PNG/JPG）、视频（MP4）、工程文件多格式导出，支持高清无损输出；

- 云端存储：自动保存工程文件，跨设备同步创作内容；

- 模板广场：用户可上传、复用节点创作模板，一键套用成熟创作逻辑。

# 四、核心技术方案（落地版本）

> ⚡ 完整技术详情请见：[[示例文档/AI无限创意画布-项目架构文档]]、[[示例文档/AI无限创意画布-技术决策记录]]

## 4.1 整体技术架构

采用**全栈 TypeScript + 三层架构**，保证轻量化、可维护、可扩展：

```
┌─────────────────────────────────────────────────┐
│  FRONTEND (React 18 + Vite)                      │
│  ┌──────────┐  ┌──────────┐  ┌───────────────┐  │
│  │Excalidraw│  │React Flow│  │ UI (Tailwind) │  │
│  │ 无限画布  │  │节点编辑器 │  │ 工具栏/侧边栏 │  │
│  │  (Fork)  │  │  (v12)   │  │               │  │
│  └──────────┘  └──────────┘  └───────────────┘  │
│  ┌─────────────────────────────────────────────┐ │
│  │   STATE (Zustand + Immer)                    │ │
│  │  画布状态 | 节点图 | AI队列 | 撤销/重做      │ │
│  └─────────────────────────────────────────────┘ │
│  ┌─────────────────────────────────────────────┐ │
│  │   STORAGE (IndexedDB / Dexie.js)             │ │
│  └─────────────────────────────────────────────┘ │
└──────────────────────┬──────────────────────────┘
                       │ HTTP / SSE
┌──────────────────────▼──────────────────────────┐
│  BACKEND (Hono + Drizzle ORM + SQLite)           │
│  ┌──────────┐  ┌──────────┐  ┌───────────────┐  │
│  │ AI Proxy │  │ 认证/项目 │  │ 文件服务      │  │
│  │ (适配器)  │  │ 管理     │  │ 上传/下载     │  │
│  └──────────┘  └──────────┘  └───────────────┘  │
└──────────────────────────────────────────────────┘
```

| 层级 | 技术选型 | 定位 |
|------|---------|------|
| **语言** | TypeScript（全栈统一） | 端到端类型安全，单人开发上下文不切换 |
| **前端框架** | React 18 + Vite | HMR极速，Excalidraw原生React生态 |
| **画布** | Excalidraw（Fork，最小侵入策略） | 只新增AI元素类型，不动核心交互逻辑 |
| **节点编辑** | React Flow (xyflow) v12 | 业界最成熟React节点编辑框架 |
| **状态管理** | Zustand + Immer | 1KB体积、无Provider、不可变state |
| **浏览器存储** | IndexedDB（Dexie.js） | 结构化查询，离线可用 |
| **后端框架** | Hono + Drizzle ORM | 超轻量，可平滑迁移Cloudflare Workers |
| **数据库** | SQLite（MVP）→ Cloudflare D1（未来） | 零运维起步 |
| **AI集成** | 适配器模式（Adapter Pattern） | 统一接口，多模型即插即用 |

> 完整技术栈清单：[[示例文档/AI无限创意画布-项目架构文档#二、技术栈详细清单]]

## 4.2 关键技术实现

### 4.2.1 无限画布渲染技术

基于 **Excalidraw 源码 Fork**，采用最小侵入策略：
- 仅新增元素类型（`AIElement`、`VideoElement`），不修改核心交互逻辑
- **分片渲染**：将画布分为 2000×2000px 虚拟Chunk，仅渲染当前视口内元素
- **LRU图片缓存**：限制最大内存占用，回收最久未使用资源
- **Web Worker**：序列化/计算在Worker线程执行，UI不阻塞

> 详情：[[示例文档/AI无限创意画布-项目架构文档#ADR-008|ADR-008 分片渲染+LRU缓存]]

### 4.2.2 几何节点适配改造

基于 **React Flow** 构建可视化节点编辑器，适配2D图文创作：
- 自定义节点类型：PromptNode / TextToImageNode / StyleNode / MergeNode 等
- **节点执行引擎自研**（核心复杂度最高的组件）：
  - 有向无环图拓扑排序 → 确定执行顺序
  - 同级节点 **Promise.all 并行执行**
  - **增量执行**：仅执行变更路径上的节点（Dirty标记）
  - 节点输出缓存：输入未变则跳过执行
- 参数面板：每个节点自带参数控件（Slider / Select / ColorPicker）

> 详情：[[示例文档/AI无限创意画布-项目架构文档#4.2 节点编辑器模块]]

### 4.2.3 AI模型融合调度

采用 **适配器模式（Adapter Pattern）**，统一接入多模型：

```
统一接口: AIProvider { textToImage, imageToImage, textToVideo, imageToVideo }
              │
              ├── OpenAIAdapter → DALL-E 3
              ├── StabilityAdapter → Stable Diffusion
              ├── ReplicateAdapter → 多模型
              ├── RunwayAdapter → 视频生成 (v0.2)
              └── MockAdapter → 离线调试
```

- 节点参数通过 **ParamMapper** 精准映射到AI模型参数
- **请求队列**（AIQueue）+ **速率限制**（RateLimiter）保证调用稳定性
- **SSE推送**：生成进度实时推送到前端

> 详情：[[示例文档/AI无限创意画布-技术决策记录#ADR-005|ADR-005 适配器模式]]

### 4.2.4 实时预览渲染技术

- 节点修改后触发增量执行，**仅重计算受影响路径**
- PreviewNode 订阅上游节点输出流，实时渲染
- 高频变更时 **debounce 节流**，避免过度渲染

## 4.3 技术壁垒

市面暂无产品实现"Excalidraw无限画布 + Blender几何节点 + 全品类AI生成"的三重融合。

本项目核心壁垒在于：
1. **轻量化节点引擎**：将Blender专业3D节点逻辑简化为2D图文/视频创作适配，保留程序化编辑的精髓，但把门槛降到普通创作者可上手
2. **适配器式AI融合**：非绑定单一模型，而是可插拔架构。未来新模型出现只需新增适配器类
3. **画布+节点+AI的三体联动**：画布上的元素可作为节点输入，节点输出可嵌入画布，形成闭环

# 十、风险评估与应对方案

## 10\.1 技术风险

**风险点**：AI生成效果不稳定、节点渲染卡顿、画布海量元素加载延迟。

**应对方案**：持续优化模型适配算法，迭代节点渲染引擎，采用分片加载、算力动态调度技术，保障产品流畅稳定；建立效果迭代机制，持续优化生成质量。

## 10\.2 市场风险

**风险点**：同类AI创意工具竞争激烈，用户获取成本高。

**应对方案**：强化产品差异化壁垒，主打“节点精细化可控\+无限自由创作”核心卖点；依托免费模式降低用户准入门槛，通过模板生态提升用户留存。

## 10\.3 合规风险

**风险点**：AI生成内容存在版权、合规风险，用户违规创作内容难以管控。

**应对方案**：搭建内容审核机制，实时筛查违规内容；明确用户创作版权协议，规避侵权风险；接入合规AI模型，保障生成内容合法合规。

## 10\.4 算力成本风险

**风险点**：AI生成算力消耗大，初期运营成本较高。

**应对方案**：优化算力调度机制，降低无效算力消耗；采用阶梯式付费模式，平衡免费用户体验与运营成本；后期通过规模化用户摊薄单用户算力成本。
