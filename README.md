# 泱泱画布（YangYang Canvas）

> **瞻彼洛矣，维水泱泱** — 《诗经・小雅》
>
> 江河浩瀚无边、广袤盛大，寓意画布容量浩瀚无穷，可承载海量创意素材、AI 生成内容与复杂节点工作流。

**泱泱画布** 是一款基于 Excalidraw 无限画布、融合 Blender 几何节点式程序化编辑逻辑、集成全品类 AI 生成能力的一站式创意设计平台。用户无需专业设计技能，即可通过节点拖拽、参数微调、图层联动，精准控制 AI 生成内容的细节、风格和动态逻辑。

---

## 产品定位

| 维度 | 说明 |
|------|------|
| **核心价值** | 在无限自由的画布上，通过拖拽节点，精准可控地完成 AI 图文创作全流程——从创意构思、素材生成、逻辑编辑、效果调试到成品输出 |
| **目标用户** | 个人创作者、自媒体从业者、中小设计团队、电商运营、学生群体 |
| **适用场景** | 图文设计、短视频创作、视觉创意、动态海报、概念设计 |
| **核心竞争力** | Excalidraw 手绘美学 + 节点式 AI 工作流 + 中文生态（国产 AI 模型优先） |

## 为什么叫"泱泱"？

> 《诗经・小雅》"瞻彼洛矣，维水泱泱"。
>
> "泱泱"本意为江河浩瀚、宽广盛大。我们以此命名，寄托三个层面的寓意：
>
> 1. **画布之泱泱** — 无限画布容量浩瀚，可容纳海量图片、长时素材、复杂节点网络
> 2. **创作之泱泱** — 创作不应被工具边界所限，AI 加持让想象力自由流淌
> 3. **格局之泱泱** — 面向中文创作者的全球化创意工具，格局开阔，海纳百川

```

Phase 6 📅 后端服务     — Hono AI 代理 API、文件上传/下载
Phase 7 📅 应用 UI      — 工具栏、侧边栏、项目管理、导出、设置
Phase 8 📅 测试与性能   — 单元测试、Mock 测试、E2E 流程测试
```

## 设计理念

### 视觉风格

参考 Figma 营销页面的设计语言（[DESIGN.md](design-md/figma/DESIGN.md)），我们追求：

- **黑白主色调**：画布纯白（`#ffffff`）、墨黑（`#000000`），让内容成为视觉主角
- **彩色区块**：使用 lime/lilac/cream/mint/pink 等柔和色块点缀，类似便签纸放置在干净桌面上
- **药丸按钮**：所有 CTA 使用 pill 形状（`rounded-full`），圆形图标按钮
- **手绘美学**：继承 Excalidraw 的手绘风格，作为产品视觉标识
- **负空间排版**：大字号 + 紧凑字距的标题，配合宽松的正文行距

### 交互模式

- **画布模式**：全屏无限画布，专注绘制和元素操作
- **节点模式**：节点编辑器叠加在画布之上，专注 AI 工作流编辑
- **一键切换**：Ctrl+Shift+N 快速切换两种模式

## 开发原则

1. **最小侵入修改** — Excalidraw fork 保持最小改动，定期（2-4 周）同步上游
2. **适配器模式** — AI 模型接入使用统一接口，新增模型只需实现适配器
3. **本地优先** — MVP 阶段数据全部存储在本地（IndexedDB），不上云
4. **Mock 优先** — AI API 调用使用 MockAdapter 开发调试，降低 API 费用
5. **单人开发** — 每次只做一件事，严格遵循 Phase Gate 工作流
