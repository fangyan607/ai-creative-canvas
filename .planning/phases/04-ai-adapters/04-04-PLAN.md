    Implementation requirements:

    **Static metadata (D-06):**
    - providerId: 'stability'
    - providerName: 'Stability AI'
    - defaultBaseUrl: 'https://api.stability.ai'

    **Constructor:** Accepts options object:
    ```typescript
    interface StabilityAdapterOptions {
      apiKey?: string

    }
    ```

    **Model-to-endpoint mapping (Pitfall 2 mitigation):**
    Define a private dispatch map:
    ```
    v1 (JSON) models: 'stable-diffusion-xl-1024-v1-0'
    v2beta (FormData) models: 'sd3.5-large', 'sd3.5-medium', 'sd3-large', 'sd3-medium'
    ```

    **execute() method** — dual dispatch logic:
    1. Extract `prompt`, `width` (default 1024), `height` (default 1024), `seed` (default 0 = random), `model` (default 'sd3.5-large'), `negativePrompt`, `stylePreset` from nodeData
    2. Determine API version using the model-to-endpoint map. If model not in map, default to v2beta.
    3. Emit 'progress' with { percent: 10, stage: 'sending_request' }

    4. **v2beta path (SD3/SD3.5):**
       - Build FormData: prompt, model, aspect_ratio (derived from width/height, e.g. '1:1', '16:9'), output_format='png'
       - Add seed if > 0
       - Add negative_prompt if present
       - Add mode='image-to-image' and image file + strength if image-to-image mode (detected by checking if inputs contains an imageBlobId, which means Phase 5 passes the blob)
       - POST to `${this.baseUrl}/v2beta/stable-image/generate/sd3`
       - Headers: { 'Authorization': `Bearer ${this.apiKey}`, 'Accept': 'application/json' }
       - Body: FormData (NOT JSON — critical per Pitfall 2)
       - Parse JSON response, extract `image` (base64) and `seed`
       - Determine image dimensions from the decoded image data

    5. **v1 path (SDXL):**
       - Build JSON body:
         ```json
         {
           "text_prompts": [{ "text": prompt, "weight": 1 }],
           "cfg_scale": 7,
           "height": height,
           "width": width,
           "samples": 1,
           "steps": 30,
           "seed": seed > 0 ? seed : undefined
         }
         ```
       - Add `style_preset` if provided
       - POST to `${this.baseUrl}/v1/generation/${engineId}/text-to-image`
       - Headers: { 'Content-Type': 'application/json', 'Accept': 'application/json', 'Authorization': `Bearer ${this.apiKey}` }
       - Parse JSON response, extract `artifacts[0].base64` and `artifacts[0].seed`
       - Engine ID comes from the model lookup

    6. Emit 'progress' with { percent: 50, stage: 'processing' }

    7. Error handling (same pattern as OpenAI adapter):
       - 401: throw AiAdapterError('auth_failed', 'Invalid API key or authentication failed')
       - 429: throw AiAdapterError('rate_limited', 'API rate limit exceeded')
       - 400/415: throw AiAdapterError('invalid_params', sanitized message)
       - Other: throw AiAdapterError('server_error', sanitized message)
       - Strip all sensitive content from error messages per Pitfall 4

    8. Convert base64 image to Blob
    9. If onStoreImage callback provided, call with blob
    10. Determine actual width/height from image data (parse PNG header if needed, or use requested dimensions as fallback)
    11. Emit 'progress' with { percent: 90, stage: 'finalizing' }
    12. Record timing with performance.now()
    13. Emit 'done' with AdapterResult
    14. Return AdapterResult with imageBlobId, width, height, seed (from response — Stability supports seeds per Pitfall 2), model, timing

    **testConnection() method:**
    - GET `${this.baseUrl}/v1/user/account` with Authorization header
    - 200 → success; 401 → failure; network error → failure with appropriate message

    **getModels() method:**
    Return array with both v1 and v2beta models:
    ```typescript
    [
      { id: 'sd3.5-large', name: 'Stable Diffusion 3.5 Large', supportedSizes: ['1024x1024', '1024x1792', '1792x1024', '1152x1152', '1152x896', '896x1152', '1344x768', '768x1344', '1536x640', '640x1536'], maxDimensions: { width: 2048, height: 2048 }, supportsImageToImage: true, supportsSeed: true },
      { id: 'sd3.5-medium', name: 'Stable Diffusion 3.5 Medium', supportedSizes: ['1024x1024', '1024x1792', '1792x1024'], maxDimensions: { width: 2048, height: 2048 }, supportsImageToImage: true, supportsSeed: true },
      { id: 'stable-diffusion-xl-1024-v1-0', name: 'SDXL 1.0', supportedSizes: ['1024x1024'], maxDimensions: { width: 1024, height: 1024 }, supportsImageToImage: true, supportsSeed: true },
    ]
    ```

    **getConfigSchema() method:**
    Return:
    ```typescript
    [
      { key: 'apiKey', label: 'API Key', type: 'password', required: true, placeholder: 'sk-...' },
      { key: 'baseUrl', label: 'Base URL', type: 'text', required: false, defaultValue: 'https://api.stability.ai', placeholder: 'https://api.stability.ai' },
    ]
    ```
