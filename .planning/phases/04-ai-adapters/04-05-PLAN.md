<objective>
Implement the ProviderStore — a BYOK (Bring Your Own Key) configuration system that stores AI provider API keys securely in IndexedDB with Web Crypto AES-256-GCM encryption. Supports per-provider custom base URLs, key rotation via versioned keys, and schema validation.

Purpose: Users can configure their own API keys and base URLs for each AI provider through auto-generated forms driven by `getConfigSchema()`. Keys are never stored in plaintext (avoiding localStorage XSS vulnerabilities). The store provides the `testConnection()` integration point so users get immediate feedback on key validity.

    2. **Create `packages/ai-core/src/config/encryption.ts`** — Web Crypto encryption utilities per D-09:

    Implementation using app-level key approach (per D-09 planner discretion: app-level key for MVP velocity, upgradable to passphrase-based later by changing only `getEncryptionKey()`):

    ```typescript
    // Encryption key version — stored alongside ciphertext for rotation support
    const KEY_VERSION = 'v0.1'
    const APP_KEY_BASE = 'ac-canvas-ai-core-encryption-v0.1'
    const PBKDF2_ITERATIONS = 600000  // OWASP recommended minimum
    const SALT = new TextEncoder().encode('ai-canvas-provider-key-salt')

    /**
     * Derive an AES-256-GCM encryption key from the app-level secret.
     * This is the only function that needs to change when upgrading to
     * user-passphrase-based encryption (per D-09 Claude's discretion note).
     */
    async function deriveEncryptionKey(): Promise<CryptoKey> {
      const enc = new TextEncoder()
      const keyMaterial = await crypto.subtle.importKey(
        'raw',
        enc.encode(APP_KEY_BASE),
        { name: 'PBKDF2' },
        false,
        ['deriveKey'],
      )
      return crypto.subtle.deriveKey(
        {
          name: 'PBKDF2',
          salt: SALT,
          iterations: PBKDF2_ITERATIONS,
          hash: 'SHA-256',
        },
        keyMaterial,
        { name: 'AES-GCM', length: 256 },
        false,
        ['encrypt', 'decrypt'],
      )
    }

    export interface EncryptedData {
      ciphertext: string   // base64-encoded
      iv: string           // base64-encoded
      keyVersion: string   // for rotation support (Pitfall 6)
    }

    /** Encrypt an API key string. Returns base64-encoded ciphertext + IV. */
    export async function encryptApiKey(apiKey: string): Promise<EncryptedData> {
      const key = await deriveEncryptionKey()
      const iv = crypto.getRandomValues(new Uint8Array(12))
      const ciphertext = await crypto.subtle.encrypt(
        { name: 'AES-GCM', iv },
        key,
        new TextEncoder().encode(apiKey),
      )
      return {
        ciphertext: base64Encode(new Uint8Array(ciphertext)),
        iv: base64Encode(iv),
        keyVersion: KEY_VERSION,
      }
    }

    /** Decrypt an API key from base64-encoded ciphertext + IV. */
    export async function decryptApiKey(
      ciphertext: string,
      iv: string,
      keyVersion: string,
    ): Promise<string> {
      // Future: check keyVersion and use appropriate key derivation
      // For v0.1, all keys use the same derivation
      const key = await deriveEncryptionKey()
      const decrypted = await crypto.subtle.decrypt(
        { name: 'AES-GCM', iv: base64Decode(iv) },
        key,
        base64Decode(ciphertext),
      )
      return new TextDecoder().decode(decrypted)
    }

    // --- Base64 helpers ---
    function base64Encode(bytes: Uint8Array): string {
      return btoa(String.fromCharCode(...bytes))
    }

    function base64Decode(base64: string): Uint8Array {
      return Uint8Array.from(atob(base64), (c) => c.charCodeAt(0))
    }
    ```

    Key design decisions per RESEARCH.md:
    - App-level encryption key (not passphrase-based) for MVP simplicity (D-09 note)
    - PBKDF2 with 600K iterations per OWASP minimum (A4 in Assumptions Log — add loading state if slow)
    - AES-256-GCM with 12-byte random IV
    - Versioned key scheme per Pitfall 6 — `keyVersion` stored alongside ciphertext for rotation support
    - All base64 encoding uses standard btoa/atob (not base64url — IndexedDB can store standard base64)

    ```typescript
    import { db, type ProviderConfigRecord } from '@ac-canvas/web/indexedb/db'
    import { encryptApiKey, decryptApiKey } from './encryption'
    import type { ProviderConfig, ConfigField } from '../interfaces/types'

    // The store operates on ProviderConfig (the unencrypted domain model)
    // Internally it maps to/from ProviderConfigRecord (the encrypted persistence model)

    export interface ProviderConfigInput {
      providerId: string
      apiKey: string           // Plaintext — encrypted before storage
      baseUrl: string
      selectedModel?: string
      isEnabled?: boolean
    }

    export class ProviderStore {
      /**
       * Save or update a provider configuration.
       * Encrypts the API key before writing to IndexedDB.
       * Validates baseUrl format before saving (D-10).
       */
      async saveConfig(input: ProviderConfigInput): Promise<void> {
        this.validateBaseUrl(input.baseUrl)
        const encrypted = await encryptApiKey(input.apiKey)
        const record: ProviderConfigRecord = {
          providerId: input.providerId,
          encryptedApiKey: encrypted.ciphertext,
          encryptionIv: encrypted.iv,
          keyVersion: encrypted.keyVersion,
          baseUrl: input.baseUrl,
          selectedModel: input.selectedModel,
          isEnabled: input.isEnabled ?? true,
          updatedAt: Date.now(),
        }
        await db.providerConfigs.put(record)
      }

      /**
       * Load a provider configuration, decrypting the API key.
       * Returns null if no config exists for this provider.
       */
      async loadConfig(providerId: string): Promise<ProviderConfig | null> {
        const record = await db.providerConfigs.get(providerId)
        if (!record) return null
        try {
          const apiKey = await decryptApiKey(
            record.encryptedApiKey,
            record.encryptionIv,
            record.keyVersion,
          )
          return {
            providerId: record.providerId,
            apiKeyEncrypted: record.encryptedApiKey,
            apiKeyIv: record.encryptionIv,
            keyVersion: record.keyVersion,
            baseUrl: record.baseUrl,
            selectedModel: record.selectedModel,
            isEnabled: record.isEnabled,
            updatedAt: record.updatedAt,
          }
        } catch {
          // Decryption failed — likely due to key rotation (Pitfall 6)
          // Return partial config without the decrypted key
          return {
            providerId: record.providerId,
            apiKeyEncrypted: record.encryptedApiKey,
            apiKeyIv: record.encryptionIv,
            keyVersion: record.keyVersion,
            baseUrl: record.baseUrl,
            selectedModel: record.selectedModel,
            isEnabled: false,   // Disable on decryption failure
            updatedAt: record.updatedAt,
          }
        }
      }

      /**
       * Get the decrypted API key for a provider.
       * Used by Phase 5 bridge when executing adapter calls.
       * Returns null if no config exists or decryption fails.
       */
      async getApiKey(providerId: string): Promise<string | null> {
        const record = await db.providerConfigs.get(providerId)
        if (!record) return null
        try {
          return await decryptApiKey(
            record.encryptedApiKey,
            record.encryptionIv,
            record.keyVersion,
          )
        } catch {
          return null
        }
      }

      /**
       * Get the base URL for a provider. Falls back to default if not configured.
       */
      async getBaseUrl(providerId: string, defaultBaseUrl: string): Promise<string> {
        const record = await db.providerConfigs.get(providerId)
        return record?.baseUrl || defaultBaseUrl
      }

      /** List all configured providers (ids only). */
      async listProviders(): Promise<string[]> {
        const records = await db.providerConfigs.toArray()
        return records.map((r) => r.providerId)
      }

      /** Delete a provider configuration. */
      async deleteConfig(providerId: string): Promise<void> {
        await db.providerConfigs.delete(providerId)
      }

      /** Check if a provider has a stored config. */
      async hasConfig(providerId: string): Promise<boolean> {
        const record = await db.providerConfigs.get(providerId)
        return !!record
      }

      /**
       * Validate baseUrl format per D-10.
       * Must be a valid URL starting with http:// or https://.
       */
      private validateBaseUrl(url: string): void {
        if (!url.startsWith('http://') && !url.startsWith('https://')) {
          throw new Error(`Invalid base URL: "${url}". Must start with http:// or https://`)
        }
        try {
          new URL(url)
        } catch {
          throw new Error(`Invalid base URL format: "${url}"`)
        }
      }
    }

    // Singleton export
    export const providerStore = new ProviderStore()
    ```
