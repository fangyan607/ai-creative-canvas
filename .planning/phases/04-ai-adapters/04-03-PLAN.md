    **Static metadata (D-06):**
    - providerId: 'openai'
    - providerName: 'OpenAI'
    - defaultBaseUrl: 'https://api.openai.com'

    **Constructor:** Accepts an options object:
    ```typescript
    interface OpenAiAdapterOptions {
      apiKey?: string
      baseUrl?: string
    }
    ```
    Store apiKey and baseUrl (defaults to this.defaultBaseUrl) as private fields.

    **execute() method** — per D-02, D-03, D-04:
    1. Extract `prompt`, `width` (default 1024), `height` (default 1024), `seed` (ignored — DALL-E 3 doesn't support determinism per Pitfall 1), `quality` (default 'standard'), `style` (default 'vivid') from nodeData
    2. Emit 'progress' with { percent: 10, stage: 'sending_request' }
    3. Call fetch(`${this.baseUrl}/v1/images/generations`, {
       method: 'POST',
       headers: { 'Content-Type': 'application/json', 'Authorization': `Bearer ${this.apiKey}` },
       body: JSON.stringify({
         model: 'dall-e-3',
         prompt,
         n: 1,
         size: `${width}x${height}`,
         quality,
         style,
         response_format: 'b64_json',
       })
     })
    4. Emit 'progress' with { percent: 30, stage: 'processing' }
    5. Handle errors per D-04 and Pitfall 4:
       - 401: throw new AiAdapterError('auth_failed', 'Invalid API key or authentication failed')
       - 429: throw new AiAdapterError('rate_limited', 'API rate limit exceeded. Please wait and try again.')
       - 4xx/5xx: Extract error message from response JSON body. Strip any sensitive content (Bearer tokens, keys). Throw AiAdapterError with code 'server_error' or 'invalid_params'.
       - Network error: throw AiAdapterError('server_error', 'Unable to reach OpenAI API. Check your network connection.')
    6. Parse response JSON. Get data[0].b64_json, data[0].revised_prompt
    7. Emit 'progress' with { percent: 60, stage: 'downloading' }
    8. Convert b64_json to Blob: base64 decode → Uint8Array → Blob with type 'image/png'
    9. Determine actual dimensions from the size string (250ms wall clock check, or parse from the response if available)
    10. If onStoreImage callback provided, call it with blob to get imageBlobId
    11. Emit 'progress' with { percent: 90, stage: 'finalizing' }
    12. Record timing using performance.now() from start
    13. Emit 'done' with the AdapterResult
    14. Return AdapterResult with: imageBlobId, width, height, seed: null (explicitly null per Pitfall 1), model: 'dall-e-3', timing (ms)
    15. Store revised_prompt on the result as extra metadata (via nodeData mutation or on the result — Phase 5 bridge will handle it). For now, just make it available.

    **testConnection() method:**
    1. Call fetch(`${this.baseUrl}/v1/models`, {
       headers: { 'Authorization': `Bearer ${this.apiKey}` }
     })
    2. If 200: return { success: true, message: 'OpenAI API connection successful' }
    3. If 401: return { success: false, message: 'Invalid API key' }
    4. Other: return { success: false, message: `Connection failed: ${statusText}` }
    5. Catch network errors: return { success: false, message: 'Network error: unable to reach OpenAI API' }

    **getModels() method:**
    - Return: [{ id: 'dall-e-3', name: 'DALL-E 3', supportedSizes: ['1024x1024', '1024x1792', '1792x1024'], maxDimensions: { width: 1792, height: 1792 }, supportsImageToImage: false, supportsSeed: false }]

    **getConfigSchema() method:**
    - Return config fields for:
      - apiKey: { key: 'apiKey', label: 'API Key', type: 'password', required: true, placeholder: 'sk-...' }
      - baseUrl: { key: 'baseUrl', label: 'Base URL', type: 'text', required: false, defaultValue: 'https://api.openai.com', placeholder: 'https://api.openai.com' }

          expect(adapter.providerId).toBe('openai')
