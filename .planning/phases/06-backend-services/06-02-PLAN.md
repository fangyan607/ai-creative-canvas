      - `on(eventType: string, handler: SseEventHandler): () => void` — register handler, return unsubscribe function
      - Private `dispatch(eventType: string, payload: SseEventPayload)` — call all handlers for eventType

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-06 | Spoofing | SSEService (sseService.ts) | accept | SSE endpoint has no authentication in MVP (D-08). EventSource connects to same-origin /api/sse/progress which is only accessible via the Vite dev proxy in development or same-origin deployment in production. Acceptable for personal tool MVP. |
| T-06-07 | Information Disclosure | VITE_AI_PROXY_MODE env var | accept | The VITE_AI_PROXY_MODE compile-time env var is embedded in the frontend JS bundle (standard Vite behavior). It contains only "direct" or "proxy" — no secrets. Acceptable by design (VITE_ prefix convention means it's client-safe). |
