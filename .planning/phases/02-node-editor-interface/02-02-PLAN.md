    The store interface:
    ```typescript
    interface NodeGraphStoreState {
      nodes: NodeGraphNode[]
      edges: NodeGraphEdge[]
      selectedNodeId: string | null
      focusMode: 'canvas' | 'nodes'

      // CRUD
      addNode: (type: NodeType, position: { x: number; y: number }) => string  // returns new node id
      removeNode: (id: string) => void
      updateNodeData: (id: string, data: Partial<NodeDataUnion>) => void
      setNodePosition: (id: string, position: { x: number; y: number }) => void
      
      // Edges
      addEdge: (edge: { source: string; target: string; sourceHandle: string; targetHandle: string }) => string
      removeEdge: (id: string) => void
      
      // Selection
      selectNode: (id: string | null) => void
      clearSelection: () => void
      
      // Focus mode
      setFocusMode: (mode: 'canvas' | 'nodes') => void
      
      // Serialization
      serialize: () => NodeGraphSerialized
      loadSerialized: (state: NodeGraphSerialized) => void
    }
    ```

    Implementation details:
    1. Use `create` from `zustand` with `immer` middleware: `import { create } from 'zustand'; import { immer } from 'zustand/middleware/immer'`
    2. `addNode`: Generate id via `crypto.randomUUID()`. Create a new node with the given type. Use `nodeTypeDefinitions` from `@ac-canvas/shared` to get the default params for the node type, and create the appropriate `NodeDataUnion` discriminated object. Add to nodes array.
    3. `removeNode`: Remove the node AND any edges where source or target matches the removed id.
    4. `updateNodeData`: Immer patch on the node's data field. Must handle the discriminated union correctly — spread existing data, apply partial updates.
    5. `addEdge`: Generate edge id via `crypto.randomUUID()`. Push to edges array.
    6. `removeEdge`: Filter out edge by id.
    7. `selectNode`: Set `selectedNodeId` to the id string.
    8. `clearSelection`: Set `selectedNodeId` to null.
    9. `serialize`: Return `{ nodes: [...this.nodes], edges: [...this.edges] }`. Transient props are not stored in the node objects, so no filtering needed (unlike CanvasStore which filters isSelected/dragging/measured from Excalidraw elements).
    10. `loadSerialized`: Replace `nodes` and `edges` with the serialized values.
    11. `setFocusMode`: Set the focus mode string.
    12. `setNodePosition`: Find node by id, set position.

    Import `NodeType`, `NodeGraphNode`, `NodeGraphEdge`, `NodeGraphSerialized`, `NodeDataUnion`, `nodeTypeDefinitions` from `@ac-canvas/shared`.

    After writing the new store, DELETE the stub file at `apps/web/src/stores/stubs/nodeGraphStore.ts`. Remove or comment out any imports from it elsewhere (check with grep).
