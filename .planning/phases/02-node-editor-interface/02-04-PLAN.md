<objective>
Build the interactive UI components: the transparent React Flow overlay, focus mode toggle, dynamic property panel, custom connection line, and template selection dialog.

Purpose: These components form the visible interactive layer of the node editor. The overlay positions React Flow on top of the Excalidraw canvas. The focus toggle lets users switch between drawing and node editing modes. The PropertyPanel provides per-node-type parameter editing. Templates give users a quick start with pre-configured node workflows.

Output: NodeEditorOverlay, FocusModeToggle, ConnectionLine, PropertyPanel, TemplateDialog, and template data constants.
</objective>

    Create three components:

    **ConnectionLine.tsx**: Custom React Flow connection line component. Receives `fromX, fromY, toX, toY, connectionStatus`. Renders a `<path>` bezier curve. When `connectionStatus === 'invalid'`, stroke is `var(--color-connection-invalid)` (red #ef4444). Otherwise stroke is `var(--color-connection-valid)` (green #1ea64a). Stroke width 2. Per D-36.

    **FocusModeToggle.tsx**: Segmented button group. Props: `focusMode: 'canvas' | 'nodes'`, `onChange: (mode: ...) => void`. Renders two buttons side by side: "Canvas" and "Nodes". Active button: `bg-black text-white`. Inactive button: `bg-white text-black`. Both: `rounded-pill` style, no gap/border between them. Keyboard hint text below each label. Export as default and named export.

    **NodeEditorOverlay.tsx**: The main React Flow wrapper. Props: `focusMode`, `onFocusModeChange`.

    Implementation details:
    - Wrap in `<ReactFlowProvider>` from @xyflow/react
    - Use `useNodesState` and `useEdgesState` hooks for local React Flow state
    - Register all 5 node types in a `useMemo`:
      ```ts
      const nodeTypes = useMemo(() => ({
        prompt: PromptNode, 'text-to-image': TextToImageNode,
        style: StyleNode, merge: MergeNode, preview: PreviewNode,
      }), [])
      ```
    - Use bidirectional sync: useEffect watches NodeGraphStore state and calls setNodes/setEdges; onNodesChange/onEdgesChange handlers update NodeGraphStore
    - ReactFlow component props:
      - `nodeTypes`, `connectionLineComponent={ConnectionLine}`
      - `isValidConnection={(conn) => validateConnection(conn.source, conn.target, conn.sourceHandle, conn.targetHandle, [], [])}`
      - `onNodesChange`, `onEdgesChange` both sync to store
      - `onConnect` validates and calls addEdge
      - `onNodeClick` calls selectNode
      - `onNodesDelete` shows shadcn Dialog confirmation before removing
      - `deleteKeyCode={['Backspace', 'Delete']}`
      - `defaultEdgeOptions={{ type: 'smoothstep', animated: true, style: { stroke: 'var(--color-socket-output)', strokeWidth: 2 } }}`
      - `fitView={false}`, `proOptions={{ hideAttribution: true }}`
    - Container: absolute positioned, fills parent, transparent bg. `pointer-events-auto` in node mode, `pointer-events-none opacity-40` in canvas mode.
    - Node palette toolbar: top-right area with 5 draggable node type buttons. Each shows Lucide icon from nodeTypeDefinitions lookup. On drag start, set `draggingNodeType`. On drop over ReactFlow pane, convert screen coords to flow position via `screenToFlowPosition` and call `addNode(type, position)`.
    - "Quick Start" button near toolbar opens TemplateDialog (locally managed state)
    - Node deletion confirmation: uses shadcn Dialog. Heading "Delete Node", body "This will remove the node and its connections. This action can be undone.", Confirm "Delete", Cancel "Cancel".
    - Viewport sync: useEffect reads CanvasStore viewport and calls `setViewport` on ReactFlow when focusMode changes, using coordinate transforms to align.

    After creating all three, export them from `packages/node-editor/src/index.ts`.
