<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Test runner -> Source code | Vitest executes source code in simulated environments (jsdom/node). Test files import and invoke production code. |
| Coverage instrumentation -> Source | @vitest/coverage-v8 instruments source for coverage tracking. Non-invasive (V8 native). |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-01 | Tampering | vitest.workspace.ts | accept | Workspace config references local files only. No external network calls. Read-only configuration. |
| T-08-02 | Information Disclosure | Coverage reports | accept | HTML/lcov reports generated locally. Not served over network in MVP. CI deferred to v0.2 (D-08). |
| T-08-03 | Denial of Service | Test runner | accept | Tests run on developer machine only. No production impact. Test execution is opt-in. |
</threat_model>
